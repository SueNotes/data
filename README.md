## 🌟 대시보드 특징

### 1. 히어로 섹션
- **2026학년도 1학기** 배지 표시
- 큰 제목과 그라디언트 텍스트 효과
- 핵심 통계 카드 4개:
  - 👥 총 학생 수: 90명
  - 📚 분반 수: 3개
  - 📝 평가 항목: 4개
  - 🎯 만점: 160점

### 2. 평가 개요 섹션
평가 항목별 상세 정보를 카드 형태로 표시:

| 평가 항목 | 배점 | 설명 |
|-----------|------|------|
| 📖 기말고사 | 100점 | 지필 평가를 통한 이론 지식 평가 |
| 🔧 수행평가 1 | 20점 | 실습 프로젝트 과제 |
| 💡 수행평가 2 | 20점 | 창의적 문제 해결 |
| 🤝 수행평가 3 | 20점 | 협동 학습 활동 |

### 3. 분석 결과 섹션
Python matplotlib으로 생성한 시각화 차트:

**Figure 1 - 성적 종합 분석**
- ✓ 분반별 기말고사 평균 비교
- ✓ 수행평가 1, 2, 3 분반별 비교
- ✓ 총점 분포 히스토그램
- ✓ 분반별 총점 박스플롯

**Figure 2 - 세부 분석**
- ✓ 평가항목별 전체 평균
- ✓ 기말고사 vs 수행평가 총점 상관관계
- ✓ 전체 등급 분포 (A/B/C/D/F)
- ✓ 분반별 항목별 성취도 비교

### 4. 히트맵 섹션
- 90명 전체 학생의 평가 항목별 성취도 시각화
- 색상 범례: 🔴 낮음(0%) → 🟡 중간(50%) → 🟢 높음(100%)
- 분반별 구분 (1반: 1-30, 2반: 31-60, 3반: 61-90)

### 5. 분석 요약 섹션
- **전체 통계**: 분반 수, 평가 항목, 총점 구성
- **등급 기준**: A(140↑), B(120↑), C(100↑), D(80↑), F(80↓)
- **주요 인사이트**: 분반별 편차, 상관관계, 개인별 강점/약점
