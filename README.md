## 📁 파일 형식

### CSV 파일 구조
```
반,번호,이름,기말고사(100점),수행1(20점),수행2(20점),수행3(20점),총점(100점)
1반,1,학생1-01,87,18,16,19,85.32
1반,2,학생1-02,92,20,19,18,90.65
...
```

### 컬럼 설명
- **반**: 학급 (1반 ~ 5반)
- **번호**: 출석번호 (1 ~ 30)
- **이름**: 학생 이름 (학생X-XX 형식)
- **기말고사(100점)**: 기말고사 점수 (40~100점 범위)
- **수행1(20점)**: 수행평가 1 점수 (10~20점 범위)
- **수행2(20점)**: 수행평가 2 점수 (10~20점 범위)
- **수행3(20점)**: 수행평가 3 점수 (10~20점 범위)
- **총점(100점)**: 최종 총점 (소수점 둘째 자리까지)

## 💡 활용 방법

### Excel에서 열기
1. `tech_scores.csv` 파일을 Excel로 열기
2. 데이터 탭 → "텍스트 나누기" 기능 사용 (필요시)
3. 구분 기호: 쉼표(,) 선택
4. 인코딩: UTF-8 선택

### Google 스프레드시트에서 열기
1. Google 스프레드시트 접속
2. 파일 → 가져오기
3. `tech_scores.csv` 업로드
4. 구분 기호: 쉼표 선택

### Python으로 분석하기
```python
import pandas as pd

# CSV 파일 읽기
df = pd.read_csv('tech_scores.csv', encoding='utf-8-sig')

# 반별 평균 계산
class_avg = df.groupby('반')['총점(100점)'].mean()
print(class_avg)

# 상위 10명 출력
top10 = df.nlargest(10, '총점(100점)')
print(top10)
```
