# 파이썬 기초 퀴즈 - 정답 및 해설

## 📝 퀴즈 문제 및 정답

### 문제 1: 변수와 데이터 타입
**질문**: 다음 중 파이썬의 기본 데이터 타입이 아닌 것은?
- A) int (정수)
- B) float (실수)
- C) char (문자) ✅ **정답**
- D) str (문자열)

**해설**: 파이썬에는 char 타입이 없습니다. 단일 문자도 str 타입으로 표현됩니다.

---

### 문제 2: 리스트 인덱싱
**질문**: 리스트 `numbers = [10, 20, 30, 40, 50]`에서 `numbers[-1]`의 값은?
- A) 10
- B) 20
- C) 40
- D) 50 ✅ **정답**

**해설**: 음수 인덱스는 뒤에서부터 세며, -1은 마지막 요소를 의미합니다.

---

### 문제 3: 조건문
**질문**: 다음 코드의 출력 결과는?
```python
x = 15
if x > 20:
    print('A')
elif x > 10:
    print('B')
else:
    print('C')
```
- A) A
- B) B ✅ **정답**
- C) C
- D) 아무것도 출력되지 않음

**해설**: x=15는 첫 번째 조건(x>20)은 거짓이지만, 두 번째 조건(x>10)은 참이므로 'B'가 출력됩니다.

---

### 문제 4: for 반복문
**질문**: 다음 코드의 출력 결과는?
```python
for i in range(3):
    print(i, end=' ')
```
- A) 1 2 3
- B) 0 1 2 ✅ **정답**
- C) 0 1 2 3
- D) 1 2

**해설**: `range(3)`은 0부터 2까지(3 미포함)의 숫자를 생성합니다.

---

### 문제 5: 함수
**질문**: 다음 함수의 반환값은?
```python
def calculate(a, b=5):
    return a + b

result = calculate(10)
```
- A) 10
- B) 15 ✅ **정답**
- C) 5
- D) 오류 발생

**해설**: b의 기본값이 5이므로, `calculate(10)`은 10 + 5 = 15를 반환합니다.

---

### 문제 6: 리스트 메서드
**질문**: 다음 코드 실행 후 fruits의 값은?
```python
fruits = ['사과', '바나나']
fruits.append('오렌지')
fruits.insert(0, '딸기')
```
- A) ['사과', '바나나', '오렌지', '딸기']
- B) ['딸기', '사과', '바나나', '오렌지'] ✅ **정답**
- C) ['사과', '딸기', '바나나', '오렌지']
- D) ['오렌지', '딸기', '사과', '바나나']

**해설**: `append`는 끝에 추가, `insert(0, '딸기')`는 0번 인덱스(맨 앞)에 '딸기'를 삽입합니다.

---

### 문제 7: 딕셔너리
**질문**: 다음 코드의 출력 결과는?
```python
person = {'이름': '홍길동', '나이': 25}
print(person['나이'])
```
- A) '나이'
- B) 25 ✅ **정답**
- C) {'나이': 25}
- D) 오류 발생

**해설**: 딕셔너리에서 키로 값을 조회하면 해당 키의 값이 반환됩니다.

---

### 문제 8: while 반복문과 break
**질문**: 다음 코드는 몇 번 반복될까요?
```python
count = 0
while count < 10:
    count += 1
    if count == 5:
        break
```
- A) 4번
- B) 5번 ✅ **정답**
- C) 9번
- D) 10번

**해설**: count가 1, 2, 3, 4, 5가 될 때까지 5번 반복되고, count==5일 때 break로 종료됩니다.

---

### 문제 9: 리스트 컴프리헨션
**질문**: 다음 리스트 컴프리헨션의 결과는?
```python
result = [x * 2 for x in range(5)]
```
- A) [0, 1, 2, 3, 4]
- B) [2, 4, 6, 8, 10]
- C) [0, 2, 4, 6, 8] ✅ **정답**
- D) [1, 2, 3, 4, 5]

**해설**: `range(5)`는 0~4를 생성하고, 각 값에 2를 곱하므로 [0, 2, 4, 6, 8]이 됩니다.

---

### 문제 10: 예외 처리
**질문**: 다음 중 0으로 나눌 때 발생하는 예외는?
- A) ValueError
- B) TypeError
- C) ZeroDivisionError ✅ **정답**
- D) IndexError

**해설**: 0으로 나눌 때는 `ZeroDivisionError`가 발생합니다.
